== Installation ==

1. Install Infugrator either via the WordPress.org plugin repository or by uploading the files to your server. (See instructions on <a href="http://www.wpbeginner.com/beginners-guide/step-by-step-guide-to-install-a-wordpress-plugin-for-beginners/" rel="friend">how to install a WordPress plugin</a>)
2. Activate Infugrator.
3. Go to <strong>Settings</strong> and complete all <strong>Infusionsoft Details</strong>
4. Navigate through the available modules and activate what you need.
